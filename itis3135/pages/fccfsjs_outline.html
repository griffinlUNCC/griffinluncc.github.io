JavaScript Curriculum

Week Number Course Title Num of Assignments Course Description
1 Code Editors 5 This portion goes over working with code editors and IDEs. It covers concepts about VS Code and its installation, useful shortcuts, and extensions.
1 Variables and Strings 62 This portion introduces you to declaring constant and standard variables, as well as working with the different data types within Javascript. You explore writing, modifying, and printing strings, as well as various forms of concatenation.
1 Booleans and Numbers 37 In this portion, you explore the basics of boolean operations and numerical operations within Javascript. You work with various addition and subtraction operations, and explore their usecases. You also cover Unary/Bitwise conditions, the basics of boolean conditionals and comparisons, and explore the crossover of booleans and numbers with falsey and truthy operations.
2 Functions 38 You learn how to implement reusable code by declaring and calling functions. You cover the syntax used to declare and call functions, as well as how to pass in arguments and use them within your function. Finally, you go over returns and their default values.
2 Arrays 35 You learn what an array is and how it can be utilized within Javascript. You learn how to declare them, various array-related operations and methods, and the best usecases for arrays.
2 Objects 28 In this section, you are introduced to Javascript objects. You learn how to create and modify these objects and their properties, distinguish between primitive and non-primitive data types, as well as methods vs functions. Finally, you learn how to parse JSON and its relationship with JS Objects.
3 Loops 22 In this section, you are introduced to loops within Javascript. You review and implement the different types of loops, such as For, for-in, for-of, while, and do-while loops.
3 Javascript Fundamentals Review 30 In this section, you review and implement all of the previously discussed topics together. This provides numerous small projects that allow you to ensure you have a strong grasp on these previous concepts.
3 Higher Order Functions and Callbacks 37 This section introduces both higher order functions and callbacks. It explains their importance and usecases within javascript, as well as various examples, such as map, filter, for-each, every, and some.
4 DOM Manipulation and Events 87 This section introduces DOM manipulation, and explains its usecase and importance within Javascript. You learn how to target and change elements, as well as add further interactivity to webpages via the DOM.
4 JavaScript and Accessibility 42 This section discusses maintaining accessibility when adding dynamic content. It covers both how this accessibility is added (ARIA States) and the important standards regarding it.
4 Debugging 8 This section discusses debugging Javascript. It goes over common errors and how to address them by both prevention and "catching" said errors with Try/Catch. It also introduces utilizing a debugger.
5 Basic Regex 47 This portion introduces "Regex", also known as Regular Expressions. It covers how these expressions are used to sort and parse data, and various methods that can be utilized to do so, including modifiers, character classes, lookaheads, and quantifiers.
5 Form Validation 103 This section introduces further form validation by using JavaScript. It discusses the importance of validating forms separately, and the methods to do so, such as preventing default submission and displaying error messages.
5 Dates 5 This section introduces the Date object. It discusses the methods used to interact with and modify these objects, as well as the usecases for these methods and formatting.
6 Audio and Video Events 55 This section introduces both audio and video elements. It explains the Audio and Video object, as well as their constructors and methods. It also covers the HTMLMediaElement API and its usecase.
6 Maps and Sets 37 This section covers both Maps and Sets, and their usecases. It also discusses variations of maps and sets such as WeakSets and WeakMaps.
6 localStorage and CRUD Operations 78 This section delves into localStorage and CRUD (Create/Read/Update/Delete). It explains what localStorage is, its standard usecases, and how it relates to sessionStorage, cookies, Cache API, and Index DB.
7 Classes 68 This section introduces classes and their related functionality. They discuss the creation of classes, inheritance, the 'this' keyword, static properties, and methods within classes.
7 Recursion 112 This section covers recursion within JavaScript. It talks about the call stack, best usecases of recursion, as well as the potential risks of it.
7 Functional Programming 23 This portion covers functional programming and what it means within Javascript. It introduces terms such as currying, nesting functions, and function purity vs impurity.
8 Asynchronous JavaScript 36 This portion introduces readers to asynchronous programming. You learn about differences between synchronous and asynchronous programming, and the various keywords used to implement it. You also cover relevant functions that use async, such as the Fetch API.

8 React Fundamentals 37 This portion introduces the ideas about libraries and frameworks. It discusses both the benefits of these frameworks and the issues that may arise from them. It also introduces the concept of a Single Page Application, and how react ties into all of the above.
